SYSTEM FEATURES

Everything needed to monitor jeepney capacity safely and in real time

🚌
3 Live Animated Jeepneys

All three jeepneys move simultaneously on the map at different route positions with smooth rotation and real-time direction facing.

📍
Your GPS Location

Geolocation API shows your exact position on the map as a blue pulsing dot. See how far you are from the next jeepney stop.

🚦
3-Tier Capacity Indicators

Green (Available 0–14), Yellow (Almost Full 15–19), Red (Full 20) — with animated progress bars and map popups.

🗺️
Leaflet + OpenStreetMap

Interactive map with stop markers, route polyline, and clickable jeepney icons. Zero paid API — 100% free.

🔔
Safety Alert System

Red warning banner triggers when any jeepney reaches full capacity of 20 passengers.

☁️
Vercel Deployment

Fully static app — no backend required. Deploy instantly to Vercel for free with zero configuration needed.
